# Frequently Asked Questions

## General

**Q: What is RAG Studio?**  
A: A platform for building RAG systems with visual design or AI automation.

**Q: Is it free?**  
A: Yes, open-source under MIT license. Cloud hosting costs apply.

**Q: Which cloud providers are supported?**  
A: AWS, GCP, Azure, and cloud-agnostic deployments.

## Designer Mode

**Q: Can I export my pipeline as code?**  
A: Yes, Python, YAML, and Terraform formats.

**Q: How accurate is the cost calculator?**  
A: ±10% accuracy based on current model pricing.

## Autopilot Mode

**Q: How long does a build take?**  
A: 5-10 minutes on average.

**Q: Can I override AI decisions?**  
A: Yes, use "Explain Decisions" to view and modify in Designer Mode.

**Q: What evaluation metrics are used?**  
A: RAGAS metrics (faithfulness, relevance, precision, recall).

## Technical

**Q: Which LLMs are supported?**  
A: OpenAI, Anthropic, Google, local models via Ollama.

**Q: Can I use my own vector database?**  
A: Yes, supports Pinecone, Weaviate, Qdrant, and more.

**Q: Is there an API?**  
A: Yes, REST API documented at /docs endpoint.
